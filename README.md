# Ready Player One

My portfolio website showcasing my work in management, software engineering, and creative problem-solving, with subtle nods to astronomy, music, and classic gaming.

## Run Locally

```bash
bundle install
bundle exec jekyll serve --livereload
```

## Jekyll Native Structure
```
scarver2.github.io/ (hyde branch)
├── _config.yml
├── _layouts/
│   ├── default.html
│   └── post.html
├── _includes/
│   ├── header.html
│   ├── footer.html
│   └── ...
├── _data/
│   └── projects.yml
├── _posts/
│   └── 2024-02-14-youtube-video.md
├── assets/
├── index.html              # Homepage
├── toolbox.md              # Auto becomes /toolbox/
├── credits.md              # Auto becomes /credits/
├── repos.md                # Auto becomes /repos/
├── rails.md                # Auto becomes /rails/
├── boss-level.md           # Auto becomes /boss-level/
├── blog.html               # Blog listing at /blog/
└── 404.html                # Custom 404
```
